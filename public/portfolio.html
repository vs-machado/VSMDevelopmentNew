Remedi

An Android application for helping users to track their medicines doses

Features

  • Schedule alarms to remind users about their medicines
  • Receive notifications if user does not mark medicine as used in the application
  • Allows users to set a treatment period or receive alarms until medicine deletion
  • Receive daily notifications after setting a pillbox refill reminder
  • List all ongoing and ended treatments
  • Warn user if he pretends to mark a medicine as used close to the next dose hour
